# 🌾 Organic vs Inorganic Crop Classification

## Overview
This project uses Sentinel-2 imagery and Random Forest to classify organic vs inorganic maize fields.

## Problem
Organic verification is manual, expensive, and not scalable.

## Solution
Satellite-based classification using vegetation indices and ML.

## Tech
- Google Earth Engine
- Sentinel-2
- Random Forest
